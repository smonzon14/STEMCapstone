Arduino
Getting Started
An Arduino is a very compact microprocessor we can use to turn code into physical projects such as robots, LCD displays, or practically any other ambition. They are relatively cheap, and come in all shapes, sizes, and processing speeds to match your project needs. Chances are, you have heard of these clever little devices as they have continued to grow in popularity. The "Hello World" program which every developer uses the first time they use an Arduino is probably blinking an LED. It is the easiest thing you can do with an Arduino without getting too technical.
Arduinos are relatively cheap. The Micro may be slightly cheaper because of it's size, but they normally range from $10-$50 depending on their capabilities. (Amazon or Ebay may have them for cheaper)
Blinking an LED
The easiest and most common example for beginners is the blinking LED program, which turns on and off an LED with set time intervals. Let's go ahead and download the software we will need to program the Arduino here. You can choose to code directly from your browser, or you could download the latest IDE software. The choice is yours.
After you open the IDE or online editor, create a new file with the following contents:
// the setup function runs once when you press reset or power the board
void setup() {
  // initialize digital pin LED_BUILTIN as an output.
  pinMode(LED_BUILTIN, OUTPUT);
}

// the loop function runs over and over again forever
void loop() {
  digitalWrite(LED_BUILTIN, HIGH);   // turn the LED on (HIGH is the voltage level)
  delay(1000);                       // wait for a second
  digitalWrite(LED_BUILTIN, LOW);    // turn the LED off by making the voltage LOW
  delay(1000);                       // wait for a second
}
Now, making sure you have selected the correct port and board name in the "Tools" section, press the upload button. In a few seconds, you should see the on-board LED blink on and off. Once you digest the code, feel free to check out the examples Arduino comes with.
ESP8266
They are super tiny and I suggest you buy them with the pins attached as they are a huge pain to solder on.
A super cheap and reliable microchip that is being widely used in smart homes and appliances is a small chip called the ESP8266. Given its size, it is capable of being both discrete and connecting to wifi or ever creating a wifi network of its own. Its versatility makes great use in projects like the one I made to detect my family's cell phones when they enter the house (Coming Soon). Again, they are super cheap, especially when you order them from AliExpress (China). They should be around $2 a piece. Once you get one, there is a lot you can do, but make sure you install the Arduino configurations from here.
